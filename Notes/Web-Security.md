# Web Security

## What is Web Security?

Web security is the practice of protecting websites, web applications, and web services from cyber attacks.

## Common Web Attacks

- SQL Injection
- Cross-Site Scripting (XSS)
- Cross-Site Request Forgery (CSRF)
- File Inclusion
- Command Injection

## Security Best Practices

- Use HTTPS
- Strong Passwords
- Input Validation
- Secure Authentication
- Regular Updates

## Tools

- Burp Suite
- OWASP ZAP
- Nikto
- Gobuster
